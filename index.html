Browser version of the classic mobile game Snake with a colorful twist

  • Built using JavaScript, jQuery, HTML and CSS

Chess

Terminal version of Chess built with Ruby

  • Utilizes multiple levels of class inheritance to keep code DRY
  • AI dynamically calculates moves based on a recursive algorithm looking to put opponent in check or checkmate
  • Cursor shows the possible moves of a piece upon piece selection
  • Possible moves are determined through use of custom deep duplication method
